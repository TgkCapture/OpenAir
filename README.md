# OpenAir 📡

> Open-source, cross-platform mobile streaming platform for broadcasters.

Deploy live TV, radio, VOD, and podcasts — on any infrastructure.

**Maintained by:** [TgkCapture](https://github.com/TgkCapture)
**License:** MIT

## Stack
- Mobile: Flutter (Android + iOS)
- Backend: Go (Gin)
- Streaming: Nginx-RTMP + HLS
- Database: PostgreSQL + Redis
- Admin: React.js

## Quick Start
```bash
cp .env.example .env
docker-compose up
```
